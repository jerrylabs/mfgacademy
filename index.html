Web Development for dummies

Vítejte na workshopu jak vytvořit web, letem světem kulometem. V následujících dvou hodinách probereme HTML, CSS a ždibíček JavaScriptu tak rychle,že to ani rychleji nepůjde. V případě záseků ve společném postupu se ihned hlaste a bude zjednána náprava. V případě složitějších dotazů si je prosím poznamenejte a budeme se jim věnovat na konci (spoiler: odpověď zní: "Ano, jde to udělat.")

1. Jak vytvořit HTML soubor

  1. Spusťte textový editor
  2. Napište "Hello world!"
  3. Uložte jako index.html
  4. Otevřete soubor ve webovém prohlížeči
  5. Profit!

2. Základní kostra HTML

HTML je značkovací jazyk, jehož prostřednictvím se vytváří web stránka jako DOM (document object model) strom. Jednotlivé značky se zapisujou do špičatých závorek, v případě párových značek obsahuje koncová značka navíc lomítko. Ukázka kostry HTML:

<!doctype html> <html lang="cs"> <head> <title>Titulek stránky</title> </head> <body> Text stránky </body> </html>

Kostru zkopírujte do svého HTML souboru, "Hello world!" přesuňte do <body> tagu jako text stránky. Správnost formátu vašeho HTML souboru můžete ověřit ve W3C validátoru.

Revoluční výhodou HTML je ovšem hypertext neboli odkazy. Z "Hello world!" si proto vytvoříme odkaz na vaši oblíbenou stránku, třeba Pornhub Google. Použijeme k tomu značku <a> (anchor) s parametrem href (hyperlink reference), jehož hodnota je cíl vašeho odkazu. Pokud chcete, aby se link otevíral defaultně v novém tabu, stačí přidat parametr target="_blank".

<a href="https://www.google.com" target="_blank">Hello world!</a>

3. CSS - barvičky pro váš web

CSS je jazyk pro popis vzhledu webu. Jednotlivá pravidla se skládají z tzv. selektoru, který určuje, na jaké elementy se má dané pravidlo vztahovat, a deklarací jednotlivých CSS vlastností a jejich hodnot. CSS se většinou načítají z externích souborů pomocí tagu <link>, ale lze je zapsat i přímo do HTML souboru. Do hlavičky vašeho HTML souboru můžete zkopírovat níže zmíněný kód, abyste invertovali defaultní barvy vaší stránky:

<style> body { color: white; background: black; } </style>

Hodnoty vlastností určující barvu lze zapisovat také jako rgb(0-255,0-255,0-255) či hexadecimáně např. #00aaff; viz tabulka níže. Pro inspiraci můžete využít přehled pojmenovaných barev.

4. Členění stránky do sekcí

Obsah stránky se člení do bloků, pro které pak můžeme pomocí CSS definovat jejich individuální vzhled a to včetně rozměrů a pozicování. Nejčastěji používaný element pro tento účel je blokový element <div>, který defaultně zabírá celou šířku stránky. Pro obdobné formátování v rámci textu se používá inline element <span>. Narozdíl od ostatních HTML elementů nenesou <div> ani <span> žádnou sémantickou informaci.

Pojďme nyní "Hello world!" v těle stránky nahradit strukturovaným obsahem. Lorem Ipsum vygenerujme 5 odstavců náhodného textu. Pokud se vám Lorem ipsum zdá nudné, můžeme použít Hipster Ipsum, Bacon Ipsum nebo Deadpool Ipsum.

Následně tyto odstavce rozdělme do jednotlivých <p> tagů (odstavec) a obalme tagem <div>, jemuž přidáme CSS třídu (class) s hodnotou "container". Výsledný kód by měl vypadat zhruba takto:

<div class="container"> <p> Lorem ipsum dolor sit amet... </p> <p> Fusce ac felis vitae neque tempor... </p> <p> ... </p> ... <div>

V prohlížeči to nevypadá nic moc. Zkusme do CSS přidat následující pravidla (lomítko s hvězdičkou je způsob, jak v CSS psát komentáře):

.container { /* Selektor začínající tečkou se týká všech elementů dané třídy */ width: 90%; /* Šířka */ max-width: 600px; /* Maximální šířka */ margin: auto; /* Zarovnání bloku na střed */ text-align: center; /* Zarovnání textu na střed */ }

Zkuste měnit velikosti okna prohlížeče. Právě jste vytvořili webovou stránku s fluidním designem, který se přizpůsobuje velikosti viewportu.

5. Responsivní design - breakpointy

Je trošku divné zobrazovat stejnou velikost písmenek na full HD monitoru i na mrňavém mobilu. Naštěstí CSS umožňuje nadefinovat si pravidla specificky pro jednotlivé rozměry viewportu. Zkopírujme níže uvedený kód do vaších stylů:

body { font-size: 20px; } @media(max-width: 1200px) { body { font-size: 18px; } } @media(max-width: 1000px) { body { font-size: 16px; } } @media(max-width: 700px) { body { font-size: 14px; } } @media(max-width: 500px) { body { font-size: 12px; } } Zkuste si nyní pohrát s velikostí okna vašeho prohlížeče - velikost písma by se měla přizpůsobovat! Navíc pokud budete pro rozměry dalších elementů používat jednotky rem, budou se rovněž škálovat podle těchto velikostí (rem = šířka písmene M v rootu DOM stromu), což je vyloženě husté! A pokud používáte Chrome, můžete se v Dev Tools (F12) podívat, jak váš super web vypadá v mobilu.

Abychom zamezili "chytrým" telefonům všelijak webovou stránku velikostně škálovat a dělat podobné prasečiny, je doporučováno do hlavičky v HTML přidat ještě:

<meta name="viewport" content="width=device-width,initial-scale=1">

6. Menu

Nyní trošku přitvrdíme. Každá webová stránka stejně jako každá restaurace potřebuje svoje menu. Nejprve si ovšem trochu pohrajeme se vzhledem našeho obsahu a pro každý ostavec nastavíme odlišnou barvu pozadí. Protože chceme, aby barva zasahovala až do okrajů stránky, nestačí ji nastavit přímo pro daný odstavec, ale budeme jej muset obalit.

Na to se výborně hodí tag <section>. Každý obalující <section> tag by měl mít unikátní třídu, abychom v CSS mohli nastavit pro každý odstavec jiné barvičky.

To ale ovšem není všechno - šířku sekcí omezuje <div> třídy container, proto jej smažeme a třídu raději nadělíme jednotlivým odstavcům. Váš kód by měl vypadat cca takto:

<section class="section1"> <p class="container"> Lorem ipsum dolor sit amet... </p> </section> <section class="section2"> <p class="container"> Fusce ac felis vitae neque tempor... </p> </section> <section class="section3"> <p class="container"> ... </p> ... <div>

Nyní nám už nebrání nic v tom, abychom rozjeli image naší stránky v pořádně duhovém stylu. Do CSS zkopírujme níže uvedený kód s tím, že barvy klidně přizpůsobte své momentální náladě ši sexuální orientaci.

.section1 { color: LightCyan; background: DarkMagenta; } .section2 { color: LightSalmon; background: SaddleBrown; } .section3 { color: DarkOliveGreen; background: PaleGreen; } .section4 { color: Lime; background: Black; } .section5 { color: SaddleBrown; background: LightYellow; }

Pokud se vám nelíbí černý rámeček, přidejte do CSS k body pravidlo "margin: 0;". Abychom mohli odkazovat na jednotlivé sekce, je třeba jim přidělit atribut id. Můžeme si zjednodušit práci v tom, že pouze přepíšeme atribut class na id, ale v tom případě je třeba upravit i výše uvedené CSS pravidla tak, aby místo tečky (pro třídu) obsahovaly selektory mřížku # (pro identifikátory).

A nyní už je čas na samotné menu - šup tam s ním! Před první sekci vložme:

<div class="menu"> <div class="menu-items"> <a href="#section1"> První sekce </a> <a href="#section2"> Druhá sekce </a> <a href="#section3"> Třetí sekce </a> <a href="#section4"> Čtvrtá sekce </a> <a href="#section5"> Pátá sekce </a> </div> </div>

No moc hezky to nevypadá. Pojďme přeskočit několik let praxe jako HTML kodér a okořeňme to nějakým sexy CSS...

.menu { background: black; color: white; width: 100%; position: fixed; /* Tohle "připíchne" menu, takže nebude scrollovat */ top: 0; /* Menu chceme na vrchu stránky */ } .menu-items { display: flex; /* Flexbox umožňuje řadit blokové elementy vedle sebe, kuk sem */ justify-content: center; /* Zarovnáme položky menu na střed */ } .menu a { /* Tohle je zanořený selektor, platí pro všechny odkazy uvnitř menu */ padding: 10px; } section { height: 100vh; /* Magická jednotka odpovídající 1 % výšky viewportu, existuje i vw na šířku */ display: flex; justify-content: center; align-items: center; /* Tohle vertikálně zarovná text odstavců na střed */ }

A nyní můžeme klikat o 106! Oujé!

7. Přidání javascriptu pro smoothscrolling

Chcete aby při klikání stránka "neskákala", ale smyslně popojížděla? Stáhněte si tenhle soubor (pravý klik + uložit jako) a pak ještě tady tento soubor do stejného adresáře, kde máte uložen váš index.html a na konec stránky těsně před zavírácí tag </body> zkopírujte tento kód:

<script src="jquery.min.js"></script> <script src="jquery.smoothscroll.js"></script>

8. responsivní menu s breakpointy

Tak pozor, tady začíná jít do tuhého. Nad menu přidáme:

<a href="#" class="menu-button">Menu</a>

Do CSS přidáme:

.menu-button { display: none; } @media (min-width: 751px) { .menu { display: block !important; } } @media (max-width: 750px) { .menu-button { display: block; position: fixed; top: 0; right: 0; padding: 20px; background: black; } .menu { display: none; width: 6rem; top: 55px; right: 0px; } .menu { flex-direction: column; } }

Pod skripty přidané v předešlém kroku přidáme

<script type="text/javascript"> $('.menu-button').click(function(){ $('.menu').slideToggle(); }) </script>
